## 描述

[Nest](https://github.com/nestjs/nest)框架，使用TypeScript开发

## 安装

```bash
$ yarn install
```

## 运行程序

```bash
# 开发模式
$ yarn run start

# 监听开发
$ yarn run start:dev

# 生产模式
$ yarn run start:prod
```

## 测试

```bash
# 单元测试
$ yarn run test

# e2e 测试
$ yarn run test:e2e

# test 覆盖率
$ yarn run test:cov
```
